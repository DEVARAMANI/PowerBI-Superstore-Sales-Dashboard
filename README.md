## ‚ú® Features
- **Star Schema Data Model**: Separate dimensions for Customer, Product, and Order linked to a central Sales fact table.
- **KPIs**: Total Sales, Profit, Quantity and Avg Delivery Days
- **Forecasting**: 15-day sales forecast using Power BI analytics.
- **Visuals**:
  - Sales trends (YoY comparison)
  - Regional performance (map and top states)
  - Category & Sub-Category breakdown
  - Payment mode and customer segment analysis
- **Interactivity**: Slicers for Region, Segment, Payment Mode, and Returns.
